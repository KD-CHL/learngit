# Learn Git

Git 操作练习仓库。

## 练习目录

- `basics/` — 基础操作（add, commit, status, log, diff）
- `branching/` — 分支操作（branch, checkout, merge, rebase）
- `remote/` — 远程操作（push, pull, fetch, remote）

## 常用命令速查

```bash
git status          # 查看状态
git add .           # 暂存所有更改
git commit -m "msg" # 提交
git log --oneline   # 查看提交历史
git branch          # 查看分支
git diff            # 查看差异
```
